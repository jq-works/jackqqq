# 🛠️ jq works (jackqqq.sys) — Personal Portfolio OS

**jq works** adalah website portfolio pribadi interaktif bertema **Retro OS & Neo-Brutalisme** yang dirancang layaknya desktop komputer klasik (*Digital Laboratory Desktop*). Website ini dibangun menggunakan **Next.js (App Router)**, **Tailwind CSS**, **Framer Motion**, dan terhubung langsung ke **Supabase** secara realtime.

---

## 🚀 Fitur Utama

- **Digital Desktop OS Simulator**: Tampilan antarmuka retro dengan window manager interaktif, dekorasi stiker pixel-art melayang, dan efek snappiness Neo-Brutalisme.
- **Audio Engine (Web Audio API)**: Efek suara *synthesizer* retro 8-bit instan saat Anda mengklik tombol, menyeret elemen, mengetik, atau memuat halaman (dapat di-mute lewat header).
- **Etalase Project (`ETALASE_PROJECT.EXE`)**: 
  - Penyaringan kategori proyek dinamis dan kolom pencarian teks.
  - Dukungan **Kustom Link & Ikon** (hingga 2 link per proyek) dengan ikon yang diintegrasikan langsung dari library [Pixelarticons](https://pixelarticons.com/).
  - Fitur **Pengurutan Proyek (Reorder)** instan via tombol panah (▲/▼) di panel admin.
- **Realtime Sticky-Note Guestbook**: Papan tulis interaktif di mana pengunjung dapat menulis catatan tempel (sticky note) dengan pilihan warna retro kustom. Catatan dapat diseret (*draggable*) di layar dan diperbarui secara instan ke pengunjung lain menggunakan Supabase Broadcast.
- **Realtime Like System**: Tombol like website di footer dengan counter angka yang sinkron secara realtime di semua browser yang aktif.
- **Bug Hunter Minigame**: Game mini arcade bertema berburu *bug* untuk memberikan pengalaman interaktif bagi pengunjung.
- **Dashboard Admin Lengkap**: Panel internal untuk mengelola statistik situs, kategori, item proyek (termasuk upload gambar ke Cloudinary), daftar pencapaian (achievements), dan pesan guestbook.

---

## 🛠️ Tech Stack

- **Frontend**: [Next.js 15+ (React 19)](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/)
- **Animasi & Interaktivitas**: [Framer Motion](https://www.framer.com/motion/)
- **Backend & Database**: [Supabase](https://supabase.com/) (PostgreSQL + Realtime Channels)
- **Asset/Image Storage**: [Cloudinary](https://cloudinary.com/) (Upload gambar proyek di admin)
- **Ikon**: [Pixelarticons React Wrapper](https://github.com/nclsndr/pixelarticons)
- **Audio**: HTML5 Web Audio API (Synthesizer Oscillator Generator)

## ⚙️ Konfigurasi Environment Variables (`.env.local`)

Buat file `.env.local` di direktori utama proyek Anda dan isi dengan variabel berikut:

```env
# Supabase Configuration
NEXT_PUBLIC_SUPABASE_URL=https://<your-project-reference>.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=<your-supabase-anon-key>

# Cloudinary Configuration
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=<your-cloudinary-cloud-name>
CLOUDINARY_API_KEY=<your-cloudinary-api-key>
CLOUDINARY_API_SECRET=<your-cloudinary-api-secret>
```

## 🗄️ Inisialisasi Database (Supabase SQL Schema)

Untuk mengaktifkan seluruh fitur dinamis (proyek, kategori, guestbook, like, dan prestasi), jalankan perintah SQL berikut di **SQL Editor** pada Supabase Dashboard Anda:

```sql
-- 1. TABEL KATEGORI PROYEK
CREATE TABLE public.categories (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    name TEXT NOT NULL UNIQUE,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL
);

-- 2. TABEL PROYEK (PROJECTS)
CREATE TABLE public.projects (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    title TEXT NOT NULL,
    category TEXT REFERENCES public.categories(name) ON UPDATE CASCADE,
    description TEXT NOT NULL,
    tags TEXT[] DEFAULT '{}'::text[] NOT NULL,
    github_url TEXT,
    link TEXT,
    custom_links JSONB DEFAULT '[]'::jsonb,
    color_class TEXT DEFAULT 'bg-retro-blue',
    year TEXT DEFAULT '2026',
    images TEXT[] DEFAULT '{}'::text[],
    is_featured BOOLEAN DEFAULT false,
    sort_order INT DEFAULT 0,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL
);

-- 3. TABEL PRESTASI & SERTIFIKASI (ACHIEVEMENTS)
CREATE TABLE public.achievements (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    title TEXT NOT NULL,
    type TEXT CHECK (type IN ('PRESTASI', 'SERTIFIKASI')) NOT NULL,
    institution TEXT NOT NULL,
    color_class TEXT DEFAULT 'bg-retro-lime',
    year TEXT DEFAULT '2026',
    created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL
);

-- 4. TABEL GUESTBOOK NOTES (STICKY NOTES)
CREATE TABLE public.guestbook_notes (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    name TEXT NOT NULL,
    message TEXT NOT NULL,
    color TEXT NOT NULL,
    rotation INTEGER NOT NULL,
    pos_x INTEGER NOT NULL,
    pos_y INTEGER NOT NULL,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL
);

-- 5. TABEL LIKES (COUNTER LIKE WEBSITE)
CREATE TABLE public.likes (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL
);

-- =======================================================
-- KEAMANAN & REALTIME CONFIGURATION
-- =======================================================

-- Aktifkan Row Level Security (RLS) pada tabel yang diakses publik
ALTER TABLE public.guestbook_notes ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.likes ENABLE ROW LEVEL SECURITY;

-- Policy Guestbook
CREATE POLICY "Allow public select access" ON public.guestbook_notes FOR SELECT USING (true);
CREATE POLICY "Allow public insert access" ON public.guestbook_notes FOR INSERT WITH CHECK (true);
CREATE POLICY "Allow public delete access" ON public.guestbook_notes FOR DELETE USING (true);

-- Policy Likes
CREATE POLICY "Allow public select access" ON public.likes FOR SELECT USING (true);
CREATE POLICY "Allow public insert access" ON public.likes FOR INSERT WITH CHECK (true);
CREATE POLICY "Allow public delete access" ON public.likes FOR DELETE USING (true);

-- Aktifkan Supabase Realtime untuk sinkronisasi instan
ALTER PUBLICATION supabase_realtime ADD TABLE public.guestbook_notes;
ALTER PUBLICATION supabase_realtime ADD TABLE public.likes;
```

## 🏃 Cara Menjalankan secara Lokal

1. **Install Dependensi**:
   ```bash
   npm install
   ```

2. **Jalankan Development Server**:
   ```bash
   npm run dev
   ```

3. Buka browser dan arahkan ke: `http://localhost:3000`

## 📜 Panduan Pengembangan Kode

- **Neo-Brutalist Layouts**: Gunakan border tebal (`border-[3px] border-black` / `border-4 border-black`) dan bayangan solid tanpa blur (`shadow-[4px_4px_0px_0px_rgba(0,0,0,1)]`).
- **Snappy Physics**: Nilai transisi *spring* animasi utama didefinisikan secara tersentralisasi di `lib/animations.ts` sebagai `retroSpring` untuk menjamin konsistensi visual di seluruh komponen.
- **Custom Links**: Dalam membuat link kustom untuk proyek, ikon dapat diambil dengan format kebab-case (misal `git-branch`) yang otomatis dideteksi dan diekspor dalam nama komponen PascalCase di [components/ProjectCard.tsx](file:///e:/Project/.all/jackqqq/components/ProjectCard.tsx).
